# 🧠 Smart Answer - Interview Prep Assistant 🚀

**Smart Answer** เป็นแอปพลิเคชัน Desktop บนระบบปฏิบัติการ Windows ที่ออกแบบมาเพื่อเป็นผู้ช่วยส่วนตัวในการเตรียมตัวสัมภาษณ์งานแบบ Real-time ตัวแอปพลิเคชันสามารถดักจับสัญญาณเสียงของผู้สัมภาษณ์ ไม่ว่าจะเป็นเสียงพูดรอบข้างผ่านไมโครโฟน (Microphone) หรือเสียงสัมภาษณ์ออนไลน์ผ่านโปรแกรมสนทนา (Zoom, MS Teams, Google Meet, Discord ฯลฯ) ผ่าน WASAPI Loopback แล้วทำการแปลงเสียงเป็นข้อความ พร้อมใช้พลังของโมเดล **Google Gemini AI** ในการวิเคราะห์เจตนา คัดกรองเฉพาะคำถาม และแนะนำแนวทางการตอบคำถามสัมภาษณ์งานอย่างมืออาชีพในรูปแบบโครงสร้างเชิงลึก (เช่น STAR Framework) ได้ทันที

---

## ✨ ฟีเจอร์หลัก (Key Features)

1. **🎙️ Dual Audio Input Mode**
   * **Mic Mode**: บันทึกเสียงพูดจากไมโครโฟนภายนอก
   * **System Audio Mode (WASAPI Loopback)**: ดักฟังเสียงที่ดังออกจากลำโพงคอมพิวเตอร์โดยตรง (ผ่าน `PyAudioWPatch`) ช่วยให้ดักจับเสียงของผู้สัมภาษณ์จากแอปพลิเคชันการประชุมออนไลน์ได้อย่างคมชัดและไร้เสียงรบกวนภายนอก
2. **🔇 Real-time VAD & Noise Calibration**
   * เมื่อกดเริ่มฟัง แอปพลิเคชันจะทำการวัดระดับเสียงเงียบรอบตัว (Noise Calibration) เป็นเวลา 1.2 วินาที เพื่อสร้างโปรไฟล์การตัดเสียงรบกวนและจูนระดับการเริ่มตรวจจับเสียงพูด (VAD Threshold) โดยอัตโนมัติ
   * **Auto-Trigger (VAD)**: ระบบตรวจจับการพูดอัตโนมัติ เมื่อหยุดพูดระบบจะส่งเสียงไปถอดความทันทีโดยไม่ต้องคอยกดปุ่มเริ่ม/หยุด
3. **🎵 Scipy Spectral Noise Reduction**
   * ประมวลผลเสียงด้วยอัลกอริทึม STFT Spectral Subtraction ผ่าน Scipy เพื่อกรองเสียงรบกวนพื้นหลังและเสียงฮัมจากอุปกรณ์เสียงออกก่อนส่งไปถอดความ
4. **💬 Live Speech Captioning**
   * แสดงผลข้อความถอดความคำพูดของผู้พูดแบบสดๆ บนหน้าจอขณะกำลังพูดผ่าน Google Speech Recognition API
5. **🤖 Gemini AI Structured Analysis**
   * ส่งคำถามสัมภาษณ์ไปวิเคราะห์ผ่าน **Google GenAI SDK** เพื่อคัดกรองข้อความทั่วไป (เช่น เสียงทักทาย, เสียงทดสอบไมค์) ออก และมุ่งเน้นการวิเคราะห์เฉพาะคำถามสัมภาษณ์จริง
   * วิเคราะห์ข้อมูลแบบจำแนกโครงสร้าง (Structured JSON Output via Pydantic Schema) เพื่อดึงคำตอบในมิติต่างๆ:
     * **Category**: หมวดหมู่คำถาม (Soft Skills, Technical, Behavioral, OOP, React, ฯลฯ)
     * **Focus Areas**: จุดประสงค์ที่ผู้สัมภาษณ์ใช้ถามเพื่อวัดผลผู้สมัคร
     * **Key Points**: ประเด็นสำคัญที่ควรหยิบยกมาตอบแบบย่อ
     * **STAR Framework**: โครงสร้างแนวคิดแนวทางการตอบ (Situation, Task, Action, Result) สำหรับคำถามเชิงพฤติกรรม
     * **Answer Strategy & Warnings**: กลยุทธ์และข้อควรระวังในการตอบ
     * **Example Outline**: ลำดับการเล่าเรื่องเป็นขั้นเป็นตอน
     * **Suggested Answer**: ตัวอย่างคำตอบแบบสมบูรณ์และเป็นภาษาเขียนระดับมืออาชีพในภาษาไทย
6. **📱 Mini Mode (ปักหมุดลอยบนสุด)**
   * สามารถสลับสไตล์ UI เป็นรูปแบบหน้าต่างขนาดจิ๋ว (470x200) และตั้งค่าให้ลอยอยู่ด้านบนสุดเสมอ (Always on Top) เพื่อให้อ่านแนวทางการตอบคำถามระหว่างการสัมภาษณ์จริงผ่านหน้าจอโปรแกรมอื่นได้อย่างสะดวกสบาย
7. **📂 History & Configuration System**
   * บันทึกคำถามและการวิเคราะห์ย้อนหลังในรูปแบบ Local JSON จัดหมวดหมู่ประวัติคำถาม และสามารถกลับมาเรียกดูหรือลบข้อมูลที่ไม่ต้องการออกได้ในภายหลัง

## 🚀 วิธีการติดตั้งและใช้งาน (Installation & Quickstart)

### 📋 สิ่งที่ต้องเตรียมก่อนเริ่มใช้งาน
* **Python** เวอร์ชั่น 3.9 ขึ้นไป
* ระบบปฏิบัติการ **Windows** (เพื่อรองรับการใช้งาน WASAPI loopback สำหรับอัดเสียงระบบ)
* **API Key ของ Gemini** (สมัครใช้งานฟรีได้ที่ [Google AI Studio](https://aistudio.google.com/))

### 📦 ขั้นตอนการติดตั้ง

1. **ดาวน์โหลดโปรเจกต์** ลงบนเครื่องของคุณ
2. **สร้าง Virtual Environment** (แนะนำ) และเปิดใช้งาน:
   ```powershell
   python -m venv .venv
   .venv\Scripts\activate
   ```
3. **ติดตั้งไลบรารีที่จำเป็น** ผ่าน `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```
4. **เปิดสิทธิ์การบันทึกเสียงลำโพงคอมพิวเตอร์ (สำหรับ System Audio Loopback)**
   * คลิกขวาที่ไอคอนลำโพงบริเวณมุมขวาล่างของแถบงาน Windows -> เลือก **Sound settings**
   * ไปที่ **More sound settings** -> แท็บ **Recording**
   * ตรวจสอบว่าเปิดใช้งานอุปกรณ์ **Stereo Mix** หรืออุปกรณ์ Output เริ่มต้นของคุณแล้ว เพื่อความเสถียรในการทำงานของ WASAPI Loopback บนระบบปฏิบัติการ Windows

## 💻 วิธีการรันโปรแกรม

รันไฟล์ `app.py` ด้วยคำสั่ง:
```bash
python app.py
```

### ⚙️ ขั้นตอนการเริ่มใช้งานครั้งแรก
1. เมื่อเปิดหน้าจอโปรแกรมขึ้นมาเป็นครั้งแรก หากยังไม่มี API Key ระบบจะนำคุณไปยังหน้า **Settings** โดยอัตโนมัติ
2. กรอก **Gemini API Key** ของคุณลงในช่องรับข้อมูล
3. เลือกโมเดลที่ต้องการใช้งาน (ค่าเริ่มต้นแนะนำเป็น `gemini-3.1-flash-lite` เพื่อการวิเคราะห์ที่รวดเร็วและคุ้มค่า)
4. เลือกรูปแบบ **Audio Source Type** ว่าจะใช้ไมค์ตนเอง (`mic`) หรือจับเสียงคอมพิวเตอร์สัมภาษณ์ออนไลน์ (`system`)
5. กดปุ่ม **Test API Key** เพื่อให้แน่ใจว่าเชื่อมต่อกับเซิร์ฟเวอร์สำเร็จ
6. กดเลือกปุ่ม **Save Settings**
7. สลับกลับมาที่หน้า **Dashboard** จากนั้นกดปุ่ม **Start Listening** และเริ่มสัมภาษณ์ได้ทันที!

## 📦 วิธีการสร้างไฟล์ Executable (.exe)

คุณสามารถแปลงแอปพลิเคชันนี้ให้เป็นไฟล์ `.exe` สำหรับใช้งานบน Windows ได้โดยใช้ **PyInstaller** ด้วยขั้นตอนดังนี้:

1. **ติดตั้ง PyInstaller** ใน Virtual Environment:
   ```bash
   pip install pyinstaller
   ```
2. **สร้างไฟล์ไอคอนของแอปพลิเคชัน (`app_icon.ico`)** โดยการรันสคริปต์เพื่อแปลงโลโก้สมองแบบ SVG เป็นไอคอนระบบ Windows:
   ```bash
   python generate_ico.py
   ```
3. **คอมไพล์โปรแกรมด้วย PyInstaller** โดยการสั่งรันไฟล์คอนฟิกูเรชัน `app.spec` ที่มีอยู่ในโครงการ:
   ```bash
   pyinstaller app.spec
   ```
4. เมื่อกระบวนการสร้างเสร็จสิ้น คุณจะพบไฟล์แอปพลิเคชัน `.exe` ที่พร้อมใช้งานภายในโฟลเดอร์:
   `dist/app.exe`

## 🎨 รายละเอียดเทคโนโลยีของแอปพลิเคชัน (Tech Stack)

* **GUI Framework**: [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) เพื่อรังสรรค์แอปพลิเคชัน Desktop สไตล์ Modern Dark Mode สวยงาม ทันสมัย และลื่นไหล
* **AI API Integration**: `google-genai` SDK เวอร์ชั่นล่าสุด เพื่อความเร็วและความแม่นยำในการเรียกใช้โครงสร้างข้อมูล JSON แบบ Structured Schema
* **Speech to Text**: `SpeechRecognition` ร่วมกับ Google Web Speech API เพื่อการแปลงข้อมูลเสียงพูดสดเป็นตัวหนังสือแบบไม่ต้องใช้ทรัพยากรบนเครื่องสูง
* **Audio Capture API**: `PyAudioWPatch` + `sounddevice` เพื่อให้โปรแกรมดึงคลื่นเสียงจากระบบได้โดยตรงโดยไม่เกิดการดีเลย์ หรือกินกำลังหน่วยประมวลผล (CPU)
* **Digital Signal Processing**: `scipy` (STFT/ISTFT) + `numpy` ในการแปลงสัญญาณเชิงเลขและลดสัญญาณรบกวนรอบตัว
